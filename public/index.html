Ayuda - Sintaxis del Template

Variables EJS:

  • <%= variable %> - Imprime el valor de una variable
  • <%- html %> - Imprime HTML sin escapar
  • <% codigo %> - Ejecuta código JavaScript

Funciones disponibles:

  • <%= formatCurrency(valor, 'S/.') %> - Formatea números como moneda
  • <%= formatNumber(valor, 2) %> - Formatea números con decimales
  • <%= moment(fecha).format('DD/MM/YYYY') %> - Formatea fechas

Configuración del Template (opcional):

<%# CONFIG: {
  "displayName": "Nombre del Template",
  "description": "Descripción",
  "fields": [
    {"name": "campo1", "label": "Campo 1", "type": "text", "required": true},
    {"name": "campo2", "label": "Campo 2", "type": "number"}
  ]
} %>
